Follow these steps to add a README file.
1. In the command line, navigate to the root directory of your new project and initialize the current repository.
2. Create a README file with some information about the project.
3. Check git status.
4. Stage and commit the file.
5. Push the changes to your branch.
6. git remote -v // Shows what all the repositories are there
7. git diff // This will show changes which are performed
